 # 🐦 Bird Species Observation Analysis

## 📌 Project Overview
This project analyzes bird species observations across forest and grassland habitats to understand patterns in bird distribution and activity. The analysis focuses on how environmental and seasonal factors influence bird behavior.

---

## 🎯 Objective
To perform exploratory data analysis (EDA) on bird observation data and identify the impact of habitat type, seasons, and environmental conditions on bird activity.

---

## 📊 Dataset Description
The dataset contains bird observation records including:
- Species information (Common & Scientific Name)
- Habitat type (Forest, Grassland)
- Date and time of observation
- Environmental factors (Temperature, Humidity, Wind, Sky)
- Observation details (Distance, ID Method, Flyover Observed)
- Bird count (Initial_Three_Min_Cnt)

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Google Colab
- Power BI (for dashboard visualization)

---

## 🔍 Data Processing
- Merged forest and grassland datasets
- Handled missing values and duplicates
- Converted date columns into proper format
- Created new features: Month, Year, Season

---

## 📈 Exploratory Data Analysis
The following visualizations were created:
- Monthly bird activity trends (Line chart)
- Seasonal distribution (Bar chart)
- Temperature vs bird activity (Scatter plot)
- Weather and wind impact (Bar charts)
- Bird count distribution (Box plot)

---

## 💡 Key Insights
- Bird activity is higher during the monsoon season
- June shows peak bird observations
- Moderate temperatures increase bird activity
- Low wind conditions favor bird presence
- Dataset covers May–July (partial seasonal analysis)

---

## 📌 Conclusion
The analysis shows that habitat type, seasonal variation, and environmental factors significantly influence bird activity and distribution. These insights can support biodiversity conservation and ecological research.

---

## 📎 Project Files
- `Bird_Species_Analysis.ipynb` – Colab notebook
- Dataset files (Forest & Grassland)
